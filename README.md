# E-Commerce Sales Analysis Project

## Business Problem
Analyzed online retail transaction data to identify customer segments, top-performing products, and purchasing trends to inform marketing and inventory strategies.

## Dataset
- Source: Kaggle - Online Retail Dataset
- Size: 25,900 different transactions
- Time Period: 2010-2011
- Geography: International online retailer based in the UK

1. **Customer Segmentation**: Identified 5 customer segments using RFM analysis
